# PBL IMS 簡報素材檔案說明

本資料夾提供 IMS 進銷存系統 PBL 專題簡報用素材。

## 檔案清單

- `PBL_IMS_簡報素材.md`：簡報文字素材、專題定位、技術選型、展示說明詞。
- `01_系統架構圖.mmd`：系統架構圖 Mermaid 原始檔。
- `02_功能流程圖.mmd`：使用者功能流程與庫存流程 Mermaid 原始檔。
- `03_AI協作開發流程.mmd`：AI 協作開發流程 Mermaid 原始檔。
- `04_資料表關聯圖.mmd`：資料庫 ER 圖 Mermaid 原始檔，可作為附錄。

## 使用方式

可將 `.mmd` 內容貼到支援 Mermaid 的工具中產生圖片，例如 VS Code Mermaid Preview、Mermaid Live Editor、Notion、GitHub Markdown，或簡報工具的 Mermaid 外掛。

簡報中建議保留三張主圖：系統架構圖、功能流程圖、AI 協作開發流程。資料表關聯圖可放在資料庫設計頁或附錄頁。
